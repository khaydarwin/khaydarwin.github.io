Maze Game

A simple maze game where the user has to avoid hitting any walls and enemies while collecting coins to advance to the next level. Failing to avoid walls and enemies cost 10 points and failing again at 0 points results in a loss. This game was coded using Python (specifically using the PyCharm IDE) and the pygame and gamebox libraries.
